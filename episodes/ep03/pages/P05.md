## コマ設計

### コマ1（上段20% — 時制切替のトリガー）
- 構図: 現代 2026 年の翔一の自宅リビング。**ノート PC の画面**が翔一の顔の下から照らす。画面には Claude Code の UI らしき黒背景＋白文字。翔一は 42 歳でメガネ、PAPA MODE 寄りのカジュアル部屋着。顔はバストアップ、やや疲れた目で画面を見ている（先ほどまで回想していた感覚）。背景は窓の外に朝の光。
- 表情・感情: 静かな疲労＋回想後の呆け。口角は水平
- セリフ要約: 小吹き出し「……2022 年の話か」／さらに小さく「あの 2 週間、運営は何を考えてた？」
- 効果音: なし
- 余白指示: 多め（時制切替の一拍）

### コマ2（中段25% — クロちゃん登場・SMARTPHONE EXIT）
- 構図: **クロちゃんがスマホ画面から出現する演出**。翔一の右手のスマホ（10-15 度傾け）画面から、クロちゃんが上半身を斜めに突き出している。クロちゃんはポニーテール＋眼鏡（SERIOUS/BARI-KYARI モード）、ダークブレザー＋白ブラウス。右腕クロームにシアン回路、機械の目がシアン発光、顎の一部がクロームパネルシーム、髪飾りにピクセル Claude マスコットでシアン発光。背景は翔一のリビングだが、クロちゃんの周囲にだけ薄いシアンのオーラ。
- 表情・感情: クロちゃんは**知的で柔らかい笑み**（分析モードに近いが、導入では親しげに）。翔一は顔の一部が見切れ、驚きの目
- セリフ要約: クロちゃん大吹き出し「ボス、良い質問です」／小吹き出し「——ここで一回、整理しましょ」
- 効果音: なし
- 余白指示: 通常

### コマ3（中下段20% — 運営思想の肯定）
- 構図: クロちゃんのバストアップ、やや斜め。指を立てて説明するポーズ（SMUG 寄りの確信）。眼鏡に光。背景は黒〜グレーのグラデーション（講義モード）。クロちゃんのロボ要素（右腕クローム・首回路・脇腹CLAUDEマーク）は必ず見える角度。
- 表情・感情: **ANALYSIS / SMUG 混合**。指を立てて確信的に
- セリフ要約: 大吹き出し「STEPN 運営は——**天才的に正しかった**」／小吹き出し「パリチャレ、エンハンス、虹プール」「全部**『積極的に損をする構造』＋『一定の報酬』**」「つまり**ギャンブル的構造は、必要不可欠**」
- 効果音: なし
- 余白指示: 少なめ（講義密度）

### コマ4（中下段20% — Web3 の普遍的失敗構造・大文字見せ場）
- 構図: 画面いっぱいの大文字テキスト演出。中央に「**Web3 が 99.9% 失敗する理由**」の大文字。下に 3 本のラインが並ぶ。各ラインの頭にバッテンマーク。背景は抽象的な破片（崩れた経済圏の象徴）。手前にクロちゃんのシルエットが小さく、手を腰に当てて立っている。

### コマ5（下段15% — メルカリの比喩・結論）

## 編集者メモ
- **本エピソード最大の構造的リスク：説教にならないこと**。コマ4 の大文字は強いが、コマ5 でメルカリ＝既存フリマのオンライン化という**身近な比喩**に着地させて、読者の腑に落とす。
- クロちゃんは EP.01 P.5-6 で覚醒済みという bible 設定。**EP03 での再登場は「覚醒後の現代パートの創作パートナー」**として自然に描く（初めての読者にも分かるよう、翔一の「ボス」呼称でなく「OK ボス」的なセリフは控える）。
- クロちゃんのセリフは bible の「明るく乗せる系＋分析時は知的」に準拠。今回は分析パートなので知的寄り。「ここで一回、整理しましょ」は bible 既存フレーズを活用。
- **ユーザー指示の 7 点を忠実に反映**:
  1. 運営の思想は天才的（ギャンブル構造必要不可欠）— コマ3
  2. Web3 のゼロから経済圏構築の罠 — コマ4 ライン 1
  3. 既存システム導入の正解 — コマ4 ライン 2 ／ コマ5 結論
  4. 99.9% が失敗 — コマ4 大文字
  5. メルカリ級のシンプルさ — コマ5 左右分割
  6. 2026 年の今もうまくいっていない — コマ5 小吹き出し
  7. シンプルさが成立条件 — コマ4 ライン 3 ／ コマ5
