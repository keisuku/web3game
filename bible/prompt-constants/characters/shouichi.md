# キャラクター固定プロンプト — 桜鬼翔一（主人公）

> モード名を指定すると該当パラグラフが挿入される。
> 複数モードが必要な場合は `mode: [casual, analysis]` のように配列で指定。
> 画像生成モデルは否定文を素材として拾うため、このファイルの英語記述は**肯定文のみ**で構成する。

## strict-rules（翔一が登場するページで強制適用）

```
Shouichi character consistency (all panels of this page):
- 42-year-old Japanese man with a grounded middle-aged appearance and ordinary everyday proportions.
- Rectangular black-framed glasses with a straight horizontal top edge and flat side rails; frames sit consistently across the nose.
- Short slightly messy black hair with salt-and-pepper greys scattered through.
- The same face shape, same glasses frame, same hairstyle, and same age impression in every panel.
- Glasses remain on the face in every panel.
- Default wardrobe: casual loungewear, T-shirt, or walk-casual outfit. A dark suit appears only when the panel text explicitly describes an office scene.
```

## base（すべてのモードに前置）

```
Character: Shouichi Sakuragi, a 42-year-old Japanese man. Rectangular black-framed glasses with a straight horizontal top edge. Short slightly messy black hair with salt-and-pepper greys. Middle-aged everyday face, grounded semi-realistic manga style, consistent across every panel.
```

## casual（デフォルト・自宅・散歩）

```
Wearing T-shirt, optional light jacket, jeans or loungewear. Home living room setting.
```

## papa

```
Wearing casual clothes, possibly with an apron. Tired but warm eyes. Domestic atmosphere.
```

## walk

```
T-shirt, light jacket, jeans, sneakers. Walking a Pembroke corgi on leash. Outdoor suburban path.
```

## work

```
Neat dark suit and tie, professional sharp gaze, office setting.
```

## wine

```
Casual clothes, holding a wine glass, blissful satisfied expression, relaxed setting.
```

## analysis（分析モード）

```
Bust shot, slightly front or angled. Right hand holding smartphone, left hand touching chin. Glasses lenses carry a subtle shadow. Serious focused eyes.
```

## confident（自信満々モード）

```
Bust shot front. Teeth-showing confident grin with a shrewd undertone. Glasses lenses carry a clean soft highlight (uniform reflection, not a star shape, not rays).
```

## shock-reversal（ショック反転モード・P04 用）

```
Bust shot front. Eyes wide open with natural pupil size keeping small highlights. Mouth slightly open, lips closed flat without showing teeth. The face holds a still processing-shock expression.
```

## stunned-disbelief（絶望衝撃モード・EP01-REBOOT P01 用）

```
Extreme close-up. Eyes wide open with natural pupil size keeping small highlights. Mouth half-open in stunned silence. Face fully covered in a waterfall of sweat beads (dozens of beads rendered in white outlines and black linework). Expression of absolute comprehension shock — quiet processing of something irreversibly ended.
```

## futon-blanket-stunned（布団カット・呆然モード・EP01-REBOOT P02 用）

```
Lying under a futon blanket in a dark room. Only the head and one hand visible. A smartphone in the hand emits a faint cyan glow lighting the face from below. Eyes slightly unfocused with natural pupil size keeping small highlights. Mouth slightly parted. Expression: stunned emptiness, confused but still lucid. This is a composition where the futon appears as a setting element only.
```

## shrewd-smile-deep（いかがわしい笑みの深い版）

```
Bust shot. Corner of mouth raised, eyes narrowed with amused realization. Left hand lightly touching mouth. Glasses lenses reflecting a calm soft light.
```

## back-view（後ろ姿）

```
Back shot from behind. Face turned away from the viewer. Posture upright, shoulders firm — indicating steady decision.
```

## cropped-face（見切れ演出）

```
Face is mostly out of frame — only the corner of the mouth and glasses rim are visible at the panel edge.
```

## hand-only（手のみ）

```
Hand close-up only. Face absent from panel. Hand shows subtle tremor at knuckles when specified.
```
