# キャラクター固定プロンプト — 桜鬼翔一（主人公）

> モード名を指定すると該当パラグラフが挿入される。
> 複数モードが必要な場合は `mode: [casual, analysis]` のように配列で指定。

## base（すべてのモードに前置）

```
Character: Shouichi Sakuragi, a 42-year-old Japanese man. RECTANGULAR black-framed glasses (never round, never thin). Short slightly messy black hair with grey mixed in (salt-and-pepper). Middle-aged, ordinary-looking, NOT a handsome idol. Semi-realistic manga style consistent across all panels.
```

## casual（デフォルト・自宅・散歩）

```
Wearing T-shirt, optional light jacket, jeans or loungewear. Home living room setting.
```

## papa

```
Wearing casual clothes, possibly with an apron. Tired but warm eyes. Domestic atmosphere.
```

## walk

```
T-shirt, light jacket, jeans, sneakers. Walking a Pembroke corgi on leash. Outdoor suburban path.
```

## work

```
Neat dark suit and tie, professional sharp gaze, office setting.
```

## wine

```
Casual clothes, holding a wine glass, blissful satisfied expression, relaxed setting.
```

## analysis（分析モード）

```
Bust shot, slightly front or angled. Right hand holding smartphone, left hand touching chin. Shadow on glasses lenses. Serious focused eyes.
```

## confident（自信満々モード）

```
Bust shot front. Teeth-showing confident grin with a shrewd undertone. Glasses lenses glinting with a soft highlight (NOT a star-shaped flash, NOT rays).
```

## shock-reversal（ショック反転モード・P04 用）

```
Bust shot front. Eyes wide open, pupils constricted. Mouth slightly open, no teeth. Smile has fully vanished. Face in pure processing-shock stillness.
```

## shrewd-smile-deep（いかがわしい笑みの深い版）

```
Bust shot. Corner of mouth raised, eyes narrowed with amused realization. Left hand lightly touching mouth. Glasses reflecting a calm soft light.
```

## back-view（後ろ姿）

```
Back shot from behind. Face not visible. Posture upright, shoulders firm — indicating steady decision.
```

## cropped-face（見切れ演出）

```
Face is mostly out of frame — only the corner of the mouth and glasses rim are visible at the panel edge.
```

## hand-only（手のみ）

```
Hand close-up only. Face absent from panel. Hand shows subtle tremor at knuckles (if specified).
```
