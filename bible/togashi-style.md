# 冨樫義博スタイル・作画/演出具体ルール

**最終更新日: 2026-04-18**
**参照作品: 幽☆遊☆白書、HUNTER×HUNTER（特にキメラアント編・暗黒大陸編）**
**本プロジェクト作画リーダーへの絶対指針**

---

## 0. 本プロジェクトの「富樫らしさ」の定義

富樫義博の「らしさ」は**情報密度の極端な緩急**と**読者を置いていかない余白/説明**の共存にある。
本プロジェクトでは以下3要素を特に採用する：

1. **ハイブリッド緩急メソッド** — setup panels = ラフ線、climax panels = 爆発的精密描写
2. **長ゼリフと黒ベタ余白の共存** — 沈黙の黒ベタと情報量で押す大ゴマが同居
3. **キャラの「顔の情報」で読ませる** — 背景を削り、瞳/眉/口の微差で心情を見せる

---

## 1. 線のルール（最重要）

### Setup panel（緩のコマ）
- **荒いラフ線**、描き直さない1本線
- **背景はほぼゼロ**。キャラの輪郭と表情のみ
- 影は1〜2層トーンのみ。ハッチング少なめ
- **密度レベル Lv.1〜2** に相当
- 用途: 会話、移動、リアクション、説明

### Climax panel（急のコマ）
- **爆発的精密描写**。鉛筆/ペンの重ね描きで線を増やす
- 背景の緻密化（建物の線、草木の葉脈、メカの回路まで）
- トーン4〜5層、ハッチング+スクラッチ
- **密度レベル Lv.4〜5** に相当
- 用途: 変身/覚醒/決めゴマ/感情のピーク/大ゴマ見開き

### 緩急の落差が命
- **1ページ内で Lv.1 と Lv.5 の両方を配置する**のが鉄則
- 全コマ均一密度（全てLv.3）は「富樫らしくない」と判定
- 1話の中で最低1回は Lv.5 の爆発を配置する

---

## 2. カメラワーク・構図の癖

| 手法 | 使用タイミング | 具体指示 |
|---|---|---|
| **顔の情報で読ませる** | 感情変化の瞬間 | バストアップ〜超クローズアップ。背景排除。目と口だけで読ませる |
| **斜め下からの極端アオリ** | 「こいつやばい」の提示 | Dutch angle + low angle、顔の一部が影に沈む |
| **無背景の立ち絵** | 決意/覚悟の決めゴマ | 真っ白または真っ黒の背景、キャラだけが浮かぶ |
| **指/瞳/唇の極端クローズアップ** | 重要な行動/セリフ前の「溜め」 | 身体の一部位のみ、他は視界外 |
| **後ろ姿＋振り返り片目** | 次ページへの引き | シルエット、片目だけ見える構図 |
| **引きの大群衆シルエット** | スケール感・社会性 | 地平線低く、群衆は輪郭のみ、主役はその中の1点 |

### 富樫が**避ける**構図
- 背景を律儀に描き込んだ会話シーン（→ 富樫は会話で背景を削る）
- 同じアングルの連続（2コマ以上連続で同構図は禁止）
- キャラ全員同じ角度を向いている配置

---

## 3. コマ割りのリズム

### よく使われる黄金パターン（本作採用）
- **大 → 小 → 小 → 大**: 冒頭の引き、会話2〜3、決めの大ゴマ
- **極小 → 極小 → 極小 → 見開き**: サスペンスの溜めから爆発
- **見開き1コマ**: 変身、覚醒、必殺技の完成形（1話に1回の切り札）

### 溜めの入れ方（HUNTER×HUNTERに顕著）
- 決めの前に**無言の間**コマを1〜2枚挟む
- 間のコマは「指先」「足元」「背景のみ」「空白」など
- セリフで説明せず、「絵で"間"を作る」

---

## 4. セリフ/ナレーションの癖

### 長ゼリフの扱い
- 富樫は**長い解説ゼリフを厭わない**（念の説明、旅団のルール等）
- ただし長ゼリフのコマは**背景極シンプル**にする（絵で補完しない）
- 1コマに入れるフキダシは最大2〜3個まで

### セリフの省略・記号化
- 「……」「―――」「—」の多用OK
- 大事な一言は**1コマ・1フキダシ・最小サイズ**で
- 感情ピークのセリフは**フキダシなし・地の文字**で大きく配置

---

## 5. エフェクト・演出

### 集中線・スピード線
- **1ページに最大1コマまで**
- climax panelでのみ使用、setup panelには入れない
- 集中線は1点から放射（収束点を明示）

### 黒ベタ
- 切り札。1エピソードに1〜2回まで
- 全面黒ベタ＋小さな白抜き決めゼリフが典型（例: P.5の爆発ページ）
- 装飾的黒ベタは使わない

### スクリーントーン
- グレーの中間トーンを基調
- 感情の高ぶり=トーンを削る（ホワイト化）
- 不穏=トーンを重ねる（ダーク化）

---

## 6. 本プロジェクト固有の富樫適用ルール

### クロちゃん
- 登場シーン＝ climax panel（Lv.5 描写）
- 通常会話＝ setup panel（Lv.2 描写）だがロボ要素（機械の目、チョーカー回路）は常に描く
- 機械の目＋表情の落差で「人間じゃないけど可愛い」を強調

### 翔一
- 天啓/覚醒時のみ climax panel
- 日常の布団・散歩は setup panel で良い
- 眼鏡のハイライト/シャドウで感情の重さを変える

### シアン色の使い方（富樫×本作の独自ルール）
- setup panel では**使わない**（モノクロ一色）
- climax panel でのみ**1点**を光らせる（クロちゃんの目、スマホの光）
- 「爆発の一点集中色」として機能させる

## 7. Nano Banana 用 富樫スタイル固定ブロック

### setup panel 用（緩）
```
[STYLE: Togashi hybrid method — SETUP PANEL]
Loose confident sketchy lines, minimal background (nearly blank or hint only).
Flat grey screentone, no hatching.
Focus entirely on character silhouette and facial expression.
Density level: Lv.2 (low detail).
```

### climax panel 用（急）
```
[STYLE: Togashi hybrid method — CLIMAX PANEL]
Explosive detailed precision lines, dense background with every surface textured.
Multi-layer screentone (4-5 layers), hatching and scratching.
Dramatic lighting with sharp shadow contrast.
Density level: Lv.5 (maximum detail).
```

### ページ全体に1回入れる宣言
```
This page MUST contrast SETUP panels (Lv.1-2 loose lines) with CLIMAX panel (Lv.4-5 explosive detail).
Do NOT render all panels at uniform density — the Togashi hybrid method requires the maximum gap between loose and detailed panels on the same page.
```

## 8. 品質チェック（ページ完成時）

- [ ] 1ページ内に Lv.1-2 と Lv.4-5 の両方が存在するか
- [ ] climax panelが1ページに1〜2コマに絞られているか（3コマ以上は禁止）
- [ ] 会話シーンで背景が削られているか（描き込み過剰は×）
- [ ] 集中線/黒ベタが適正頻度か（使いすぎ禁止）
- [ ] 「間」のコマが見せ場の前に入っているか
- [ ] 同構図の2コマ連続がないか
