# コマ割りパターン集

**本プロジェクトで使用する全パターンのカタログ。**
**「いつもの田の字4コマ」を避けるために、ページ設計時は必ずこのカタログから選ぶ。**

---

## 運用ルール（絶対）

1. **ページ設計時、必ずこのカタログからパターンIDを1つ選んでページ設計書の冒頭に記入する**
   - フィールド名: `採用パターン: [pattern-id]`
2. **前後5ページで連続して同じパターンを使わない**（最低5ページ間隔を空ける）
3. **1エピソード内で同じパターンの使用上限は3回まで**
4. `validate-pipeline.sh` がこの規則を自動チェックする
5. 新しいパターンを追加したら必ず `patterns.json` に登録する

## 詳細フォーマット

各パターンファイル（`pXX-*.md`）には以下が必須:
- 視線誘導（必ず**右上→左下**前提）
- コマサイズ比率
- 各コマの密度レベル（Togashi hybrid method）
- 適用シーン例
- Nano Banana用 composition プロンプト
